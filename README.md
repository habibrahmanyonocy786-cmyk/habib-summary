# Habib-summary

مرجع خلاصه کتاب و آزمون - Book Summaries & Quiz App

یک برنامه تک‌فایلی HTML/CSS/JS برای مرور خلاصه کتاب‌ها و شرکت در آزمون.

## ویژگی‌ها

- خلاصه ۳ کتاب برتر (Atomic Habits, Rich Dad Poor Dad, 7 Habits)
- سوالات چهارگزینه‌ای برای هر کتاب
- آزمون ترکیبی (۵ سوال تصادفی)
- طراحی واکنش‌گرا (موبایل و دسکتاپ)
- کاملاً آفلاین - بدون نیاز به backend

## نحوه استفاده

فایل `index.html` را در مرورگر باز کنید.
